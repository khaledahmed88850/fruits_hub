# Fruits Hub - E-Commerce App

![app_icon](https://github.com/user-attachments/assets/76e6a110-63ce-4bbe-8f2f-d7c95792391b)


Fruits Hub is a modern e-commerce application designed to bring fresh fruits to your doorstep. With a user-friendly interface and seamless navigation, Fruits Hub makes it easy to browse, order, and enjoy the freshest fruits delivered right to you.

## Features
- **User Authentication**: Secure login and registration using Email, Google, and Facebook.
- **Product Browsing**: Browse a wide variety of fresh fruits with detailed descriptions and images.
- **Search Functionality**: Easily search for specific fruits using keywords.
- **Shopping Cart**: Add and manage items in the cart before checkout.
- **Secure Checkout**: Integrated PayPal payment gateway for secure transactions.
- **Admin Dashboard**: Custom dashboard built with Supabase for product management (add, update, delete products).
- **Order History**: View past orders and track their status.

## Technologies Used

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Authentication, Cloud Firestore)
- **State Management**: Cubit
- **Payment Gateway**: PayPal API
- **Admin Dashboard**: Supabase
- **Version Control**: Git
- **Architecture**: Clean Architecture
